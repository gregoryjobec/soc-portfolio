Day 1: AWS CloudTrail Setup

WHAT I LEARNED:
1. CloudTrail is AWS's audit logging service
2. It records all API calls (events) made in AWS account
3. Events are stored in S3 bucket
4. Can be used for security monitoring and compliance

WHAT I DID:
1. Created AWS free account
2. Completed TryHackMe "Introduction to AWS" room
3. Enabled CloudTrail
4. Created test events:
   - Security group
   - IAM user
   - S3 bucket
5. Verified CloudTrail captured all events

KEY EVENTS CAPTURED:
- CreateSecurityGroup
- CreateUser
- CreateBucket
